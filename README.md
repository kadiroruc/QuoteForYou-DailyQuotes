
# QuoteForYou - DailyQuotes

A mobile app where you can view quotes on a daily basis, receive notifications and save them to your favorites.


## Tech Stack

**App**:  `Swift UIKit`

**Architecture**:  `MVVM`

**Backend**:  `API Networking, Core Data`

**UI**:  `Storyboard, Auto Layouts`
